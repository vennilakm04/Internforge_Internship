## 🚀 Projects Overview

### 🏠 House Price Prediction
- Built regression models to predict house prices based on features like area, location, and number of rooms.
- Applied data preprocessing, feature encoding, and scaling.
- Models used: Linear Regression, Decision Tree, XGBoost
- Evaluation metrics: RMSE, MAE, R²

---

### 💳 Credit Card Fraud Detection
- Developed a classification model to detect fraudulent transactions.
- Handled imbalanced data using SMOTE.
- Models used: Random Forest, XGBoost
- Applied anomaly detection using Isolation Forest.
- Evaluation metrics: Precision, Recall, F1-score, Confusion Matrix

---

### 📞 Telecom Customer Prediction
- Built a classification model to predict customer behavior (e.g., churn prediction).
- Performed data cleaning, encoding, and feature engineering.
- Applied machine learning algorithms for prediction.
- Evaluation metrics: Accuracy, Precision, Recall, F1-score
